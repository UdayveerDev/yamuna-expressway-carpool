# Contribution Guidelines

Thank you for considering contributing to the Yamuna Expressway Carpool project! We appreciate your efforts in improving our project. Consider following these guidelines:

## How to Contribute

1. **Fork the repository**: Create a personal copy of the repository by forking it.
2. **Clone the repository**: Clone your forked repository to your local machine.
   ```bash
   git clone https://github.com/UdayveerDev/yamuna-expressway-carpool.git
   ```
3. **Create a new branch**: Create a new branch to work on your changes.
   ```bash
   git checkout -b your-feature-branch
   ```
4. **Make your changes**: Implement the desired changes or features in your branch.
5. **Commit your changes**: Commit your changes with a descriptive commit message.
   ```bash
   git commit -m "Descriptive message about the changes made"
   ```
6. **Push your changes**: Push your changes to your GitHub fork.
   ```bash
   git push origin your-feature-branch
   ```
7. **Create a pull request**: Go to the original repository where you want your changes to be merged. Click on the **New Pull Request** button and submit.

## Code of Conduct

Please adhere to our [Code of Conduct](CODE_OF_CONDUCT.md).

## Questions or feedback?

If you have any questions or feedback regarding the contribution process, feel free to reach out to us! We're happy to help.

Thank you for your contributions!
