# Installation Instructions for Yamuna Expressway Carpool

## Requirements
- Node.js (v14 or later)
- npm (v6 or later)

## Steps to Set Up
1. **Clone the repository**
   ```bash
   git clone https://github.com/UdayveerDev/yamuna-expressway-carpool.git
   cd yamuna-expressway-carpool
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the application**
   To start the application, run:
   ```bash
   npm start
   ```

4. **Access the application**
   Open your browser and go to `http://localhost:3000` to access the application.

## Notes
- Make sure to configure your environment variables before running the app.
- If you face any issues, refer to the troubleshooting section in the documentation.
