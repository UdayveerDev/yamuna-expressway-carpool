# Yamuna Expressway Carpool System 

A dedicated carpooling platform designed to streamline travel along the Yamuna Expressway. This system facilitates seamless booking for customers and robust management tools for drivers, aimed at making long-distance commuting more efficient and cost-effective.
## Project Overview

Traveling on the Yamuna Expressway often involves high toll costs and long stretches of driving. This project provides a centralized platform to connect drivers with empty seats to passengers traveling along the same route.

### Key Features
* **Customer Booking:** Easy-to-use interface for passengers to find and book rides.
* **Driver Management:** Tools for drivers to list vehicles, manage schedules, and track bookings.
* **Optimized Routing:** Specifically tailored for the entry/exit points of the Yamuna Expressway.
* **Database Integration:** Scalable backend architecture using SQL.

##  Getting Started

### Prerequisites
* **Node.js** (v14 or higher)
* **MySQL** or a compatible SQL database
* **npm** or **yarn**

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/UdayveerDev/yamuna-expressway-carpool.git](https://github.com/UdayveerDev/yamuna-expressway-carpool.git)
    cd yamuna-expressway-carpool
    ```

2.  **Database Setup:**
    Import the schema to your local database:
    ```bash
    mysql -u your_username -p < schema.sql
    ```

3.  **Backend Setup:**
    ```bash
    cd backend
    npm install
    npm start
    ```

4.  **Frontend Setup:**
    ```bash
    cd ../frontend
    npm install
    npm start
    ```
